# cyberlegionnaire.com

Static site for [cyberlegionnaire.com](https://cyberlegionnaire.com) hosted on GitHub Pages.

Built with plain HTML/CSS/JS — no build step required.

## Deploy

1. Push to `main` branch on GitHub
2. Go to repo **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. (Optional) Add custom domain `cyberlegionnaire.com` and enable HTTPS
